# Prueba Esto — Ejercicio 05

1. Calcula cuántos segundos hay en un año:
   ```python
   seconds_per_minute = 60
   minutes_per_hour = 60
   hours_per_day = 24
   days_per_year = 365
   seconds_per_year = seconds_per_minute * minutes_per_hour * hours_per_day * days_per_year
   print("Segundos en un año:", seconds_per_year)
   ```

2. Usa el operador módulo `%` para verificar si un número es par o impar. Los números pares tienen un residuo de 0 cuando se dividen entre 2:
   ```python
   print(10 % 2)  # 0 significa par
   print(7 % 2)   # 1 significa impar
   ```

3. Intenta dividir entre cero: `print(10 / 0)`. Lee el mensaje de error con cuidado. Python se rehúsa a hacerlo y te dice por qué.
