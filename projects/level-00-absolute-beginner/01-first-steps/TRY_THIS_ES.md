# Prueba Esto — Ejercicio 01

1. En el modo interactivo (`python` y luego `>>>`), intenta calcular tu edad en días:
   ```
   tu_edad * 365
   ```

2. Intenta dividir números que no se dividen exactamente:
   ```
   7 / 2
   10 / 3
   ```
   Fíjate que Python te da una respuesta con decimales.

3. Prueba usar `**` para exponentes (potencias):
   ```
   2 ** 10
   3 ** 3
   ```
   `2 ** 10` significa "2 elevado a la potencia de 10", que es 1024.
