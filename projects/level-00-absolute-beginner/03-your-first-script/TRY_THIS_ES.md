# Prueba Esto — Ejercicio 03

1. Crea un archivo nuevo llamado `mi_script.py` en esta carpeta. Pon los `print()` que quieras. Ejecútalo con `python mi_script.py`.

2. Agrega un comentario (una línea que empiece con `#`) para explicar qué hace tu script. Ejecútalo de nuevo — fíjate que el comentario no tiene ningún efecto en la salida.

3. Intenta poner un `#` delante de una de las líneas de `print()`. Ejecuta el script. Esa línea no se va a ejecutar porque Python ahora piensa que es un comentario. Quita el `#` para que vuelva a funcionar.
