# Nivel 00 — Principiante Absoluto (Versiones en Español)

Inicio: [README](../../README.md) | [English version](./README.md)

Este directorio contiene traducciones al español de los 15 ejercicios del Nivel 00. Las versiones en español existen junto a los archivos originales en inglés dentro de cada carpeta de proyecto.

## Archivos traducidos

Cada ejercicio tiene dos archivos en español:

| Archivo | Descripción |
|---------|-------------|
| `exercise_es.py` | Comentarios y strings de print en español. Los nombres de variables y la lógica del código son idénticos al original. |
| `TRY_THIS_ES.md` | Traducción completa de los desafíos de extensión. |

## Convenciones de traducción

- **Nombres de variables** se mantienen en inglés (convención de Python)
- **La lógica del código** es idéntica a las versiones en inglés
- **Comentarios y strings** están en español latinoamericano conversacional
- **Términos técnicos** comúnmente usados en inglés se conservan (por ejemplo, "print", "input", "string", "list", "break", "return")
- **Imports y firmas de funciones** son idénticos

## Cómo ejecutar los ejercicios

```bash
# Navega a la carpeta del ejercicio
cd projects/level-00-absolute-beginner/01-first-steps

# Ejecuta la versión en español
python exercise_es.py
```

## Cómo trabajar cada ejercicio

1. **Lee** el archivo `exercise_es.py` — cada línea tiene un comentario explicando qué hace
2. **Ejecútalo** — `python exercise_es.py` y observa qué pasa
3. **Cambia** algo pequeño y ejecútalo de nuevo
4. **Prueba** los desafíos en `TRY_THIS_ES.md`
5. **Continúa** cuando puedas explicar qué hace el código sin leer los comentarios

## Importante

- No hay tests en este nivel. Solo ejecuta y observa.
- No hay forma incorrecta de experimentar. Rompe cosas. Así es como se aprende.
- Si algo no funciona, lee el mensaje de error. Generalmente te dice exactamente qué salió mal.
- No necesitas memorizar nada. Necesitas entender el patrón.

## Cuando termines

Estás listo para el [Nivel 0](../level-0/README.md) cuando puedas:
- Crear un archivo `.py` desde cero
- Usar variables, un ciclo y una función en el mismo script
- Leer un mensaje de error simple y arreglar el problema
- Explicar qué hace tu código en tus propias palabras
