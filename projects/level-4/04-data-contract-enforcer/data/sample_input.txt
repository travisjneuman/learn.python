Sample data for Data Contract Enforcer
----------------------------------------
This file explains the companion files.

contract.json defines the expected data contract:
  - Required columns: id (integer), name (string), email (string)
  - Optional columns: phone (string), active (boolean)
  - Uniqueness constraint on "id"
  - Format constraint: email must contain "@"
