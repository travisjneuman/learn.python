# Projects Ladder (Level 0 -> Level 10)
Home: [README](../README.md)

This folder contains guided learning projects for every skill level.
Each level has 15 projects designed for the workflow:
- alter
- break
- fix
- explain

## Levels
- [level-0](./level-0/README.md): absolute beginner
- [level-1](./level-1/README.md): beginner fundamentals
- [level-2](./level-2/README.md): beginner plus data basics
- [level-3](./level-3/README.md): early junior engineering habits
- [level-4](./level-4/README.md): robust file and data handling
- [level-5](./level-5/README.md): intermediate automation patterns
- [level-6](./level-6/README.md): SQL and pipeline discipline
- [level-7](./level-7/README.md): config-driven tools and quality depth
- [level-8](./level-8/README.md): performance and reliability patterns
- [level-9](./level-9/README.md): architecture and observability
- [level-10](./level-10/README.md): SME-level system thinking

## How to use each project
1. Read the project README.
2. Run the starter script.
3. Complete the "Alter it" tasks.
4. Perform the "Break it" drills.
5. Fix issues and record root cause.
6. Expand tests.

## Suggested mapping to roadmap phases
- Levels 0-2: setup + foundations
- Levels 3-5: quality + file/excel automation
- Levels 6-7: SQL + ingestion discipline
- Levels 8-9: Orion/DPA integration + dashboard hardening
- Level 10: capstones + handoff + SME review readiness
