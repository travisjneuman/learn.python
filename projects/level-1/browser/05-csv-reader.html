Level 1 / Project 05 — CSV First Reader Parse CSV text into a list of dicts, detect numeric columns, and compute column statistics. Browser adaptation: Instead of reading from a file with csv.DictReader, you will parse a CSV string directly. The same logic applies -- split lines, use the header row as keys, and build dictionaries for each data row. 1. parse_csv(text) - The first line is the header row (comma-separated column names) - Each subsequent line is a data row - Return a list of dicts where keys come from the header - Empty text returns an empty list 2. detect_numeric_columns(rows) - Find columns where ALL non-empty values can be converted to float - Return a list of column names that are numeric 3. column_stats(rows, column) - Compute min, max, sum, and average for a numeric column - Return: {"column": name, "count": n, "min": ..., "max": ..., "sum": ..., "average": ...} - If no values, return: {"column": name, "count": 0} Examples: csv_text = "name,age,city\nAlice,30,Denver\nBob,25,Austin" parse_csv(csv_text) -> [{"name": "Alice", "age": "30", "city": "Denver"}, ...] rows = [{"name": "A", "score": "90"}, {"name": "B", "score": "85"}] detect_numeric_columns(rows) -> ["score"] column_stats(rows, "score") -> {"column": "score", "count": 2, "min": 85.0, ...}
