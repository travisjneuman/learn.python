Level 1 / Project 02 — Password Strength Checker Score passwords based on length, character variety, and common-password checks. 1. check_length(password) - Return 0 if fewer than 8 chars, 1 if 8-11, 2 if 12-15, 3 if 16+ 2. check_character_variety(password) - Return a dict: {"uppercase": bool, "lowercase": bool, "digit": bool, "special": bool} 3. check_common(password) - Return True if password (case-insensitive) is in a common passwords list - Common list: "password", "123456", "qwerty", "abc123", "letmein", "admin", "welcome", "monkey", "dragon", "master" 4. score_password(password) - Length: 0-3 points (from check_length) - Variety: 1 point per character class present (max 4) - Not common: 1 point (0 if it IS common) - Max total: 8 points - Strength labels: 7+ = "strong", 5-6 = "moderate", 3-4 = "weak", 0-2 = "very weak" - Return dict with: password, length, total_score, strength, and more Examples: check_length("abc") -> 0 check_length("abcdefghijklmnop") -> 3 check_common("password") -> True score_password("MyStr0ng!Pass#2024")["strength"] -> "strong"
